# curriculum-databases-projects-template

> This is a clinic database. The database diagram was provided and our task was to interpret it, and implement the SQL needed to build that database.

## Getting Started

- Use [schema_based_on_diagram.sql](./schema_based_on_diagram.sql) to create all tables.
